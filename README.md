# Doorifix

Doorifix is a Next.js application for appliance repair service pages, city pages, area pages, booking leads, and SEO metadata.

## Development

```sh
npm install
npm run dev
```

## Production

```sh
npm run build
npm run start
```

## Stack

- Next.js App Router
- React
- TypeScript
- Tailwind CSS
- shadcn/ui-style component primitives
- Supabase API routes for lead capture

The site is configured for `https://doorifix.com` through the shared SEO helpers, sitemap, and robots metadata routes.
